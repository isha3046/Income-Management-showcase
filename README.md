Income & Expense Manager App
A simple and attractive Flutter app to manage daily income and expenses using SQLite database with full offline support.

Features :
Add Income & Expense with related category 
Offline SQLite Database
Dashboard with Total Balance
Charts to see the totla Income/ Expense in
Simple & Modern UI
Fast & Lightweight App
Category-wise Expense Tracking / Income Tracking

Technology Used :
Flutter
SQLite (sqflite)
Chart
 
App Purpose:
Helps users track spending, manage savings, and analyze financial data easily without internet connection.
